# Trafy Landing Page

## Running the Project

### Main Website
To view the main landing page in your browser:

```bash
npm start
```

This will start a local server at [http://localhost:3000](http://localhost:3000).

### Usage Guide (Kullanım Kılavuzu)
To develop the usage guide independently:

```bash
npm run start:guide
```

This will start a separate server at [http://localhost:3000](http://localhost:3000) serving only the usage guide.

**Note:** You can run both servers simultaneously on different ports if needed.
